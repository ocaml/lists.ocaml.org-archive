Message: bap-llvm will now have an OR dependency

bap-llvm will use conf-bap-llvm if a user didn't install explicitly
conf-llvm. This allows the following:

- `opam install bap` - will use  conf-bap-llvm
- `opam install bap conf-llvm.VER` - will force bap to use llvm-VER
- `opam depext --install bap` - will use conf-bap-llvm for depexts
- `opam depext -i conf-llvm.VER; opam depext bap` will install system
   `llvm-VER` package, and force bap to use it.
