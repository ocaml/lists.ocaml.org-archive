Slightly off topic but I thought the following was interesting. http://www.kickstarter.com/projects/homebrew/brew-test-bot

It's a kickstarter for the homebrew project [1] to set themselves up with some automated test/build equipment (i.e some Mac Minis with VMs).  It'll provide automated feedback to people who submit pull-requests, making the developers job a little easier.  Planning to launch it in April.  

The kickstarter opened today today and by the time you see this, will probably have (far) exceeded their goal.  Since I haven't seen any homebrew 'donate' buttons anywhere before this may release some pent up appreciation.  Crowd funding the machine pool is also an interesting idea.

Amir

[1] http://mxcl.github.com/homebrew/ (a package manager on the mac)
