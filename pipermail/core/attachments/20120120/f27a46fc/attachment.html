        version is:</div>
      <div><br>
      </div>
      <div>1) very similar core design</div>
      <div>2) much less C code in Async</div>
      <div>3) Monitors instead of stuffing errors in the monads</div>
      <div>4) Async has more disciplined distribution of unhandled
        exceptions, and a more disciplined interleaving model</div>
      <div>5) Lwt has more hooks into the outside world and is more
        widely used</div>
      <div><br>
      </div>
      <div>(4) is a complicated one to explain, so perhaps that's one to
