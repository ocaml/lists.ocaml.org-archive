</div>- Install Homebrew from <a href="http://github.com/mxcl/homebrew" target="_blank">http://github.com/mxcl/homebrew</a><br>
- $ brew install ocaml<br>
- $ brew tap mirage/ocaml      # hook in custom ocaml packages<br>
- $ brew install opam --HEAD   # get the very latest OPAM<br>
- $ opam install core async<br>
- $ eval `opam config -env`    # import the right PATH<br>
<br>
Some of these steps will disappear in a bit (e.g. the tap is only<br>
required until a more stable OPAM appears, which will hopefully be<br>
before ICFP in a couple of weeks).<br>
<br>
Note that Homebrew just updated to ocaml-4.00.0, so hopefully Core<br>
works with it too.<br>
