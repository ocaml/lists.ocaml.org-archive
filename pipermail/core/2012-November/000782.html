>>
>> t_of_sexp : Sexp.t -> t
>> sexp_of_t : t -> Sexp.t
>>
>> (Replace "t" with the name of your type throughout.)
>>
>> You can use these with, e.g., Writer.write_sexp(s), Reader.load_sexp(s).
