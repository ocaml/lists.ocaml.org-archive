Has anyone looked at using IOCaml, including for graphics? Lessons
could be distributed as notebooks and students could run the examples
and tests in-place. There's even a version that can run over a full
JavaScript kernel, which shouldn't need any installation on the part
of users. Of course, this is limited by what libraries will run when
compiled to JS (any lib that binds to C code is out of that set). I
feel that a bit more of infrastructure around IOCaml for teaching
could be immensely useful.
